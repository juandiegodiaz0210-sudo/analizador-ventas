# analizador-ventas

Este proyecto consiste en el desarrollo de un analizador de ventas utilizando Python, con el objetivo de identificar patrones, tendencias y métricas clave a partir de datos de un negocio.

---

##  Objetivo

Analizar datos de ventas para obtener información útil como:
- Productos más vendidos
- Periodos con mayores ventas
- Comportamiento general de las ventas

---

##  Datos

El proyecto utiliza dos archivos CSV:

- Datos_Ventas_Tienda1.csv  
- Datos_Ventas_Tienda2.csv  

Ambos archivos se combinan para realizar un análisis completo de las ventas.

---

##  Funcionalidades

- Carga y unión de múltiples archivos CSV  
- Limpieza de datos (valores nulos, formatos)  
- Análisis de ventas por producto y por mes  
- Visualización de datos mediante gráficos  
- Generación de conclusiones automáticas  

---

## Tecnologías utilizadas

- Python  
- Pandas  
- Matplotlib  

---

##  Cómo ejecutar el proyecto

1. Descargar los archivos del repositorio  
2. Abrir el notebook en Jupyter Notebook  
3. Ejecutar todas las celdas  
4. Visualizar los resultados y gráficos generados  

---

##  Resultados

El análisis permite identificar:
- El producto con mayor número de ventas  
- El mes con mayor rendimiento  
- Tendencias en el comportamiento de ventas
